== Description ==
This Google Reviews Plugin pulls reviews from Google profiles and displays them on your website.

== Installation ==
1. Install the plugin
2. Activate the plugin
3. Paste your API key from Google Developer Console
4. Paste your Places ID
5. Chose your design
6. Use the shortcode to display your reviews
